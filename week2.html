Week 2: Variables & Data

What is a Variable?

Something that varies or changes. In code, we will use variables as a tool to store values that will change, values that will be used multiple times throughout our code, or both!

Types of variables

Variables can store different types of data.

  • Integers

  • Decimals

  • Characters

  • Strings

How do we create a variable?

There are two components to naming a variable, 1. the data type and 2. your variables name. In p5 (javaScript), variables are flexible...meaning you don't need to specify what kind of data the variable will hold.

After naming your variable, you can initialize a.k.a set it equal to a value.

p5's built in variables

p5 has a number of built in variables. We've already used one! Think about how we used the 'width' and 'height' variables to position our objects on the screen.

Variable scope

Where you put your variables matter! Remember our curly brackets are like a suitcase analogy. If the item (variable declared) is inside of a suitcase (the function brackets) it will be hidden to other...suitcases. If you've created a variable, and are getting a 'variableName' does not exist error, check to make sure your variable is declared globally (accessible by the entire sketch) or in the right local location (accessible to the function).
