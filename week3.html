Week 3: Let's Get Logical

What is a conditional?

A conditional is an expression that evaluates to either true or false; they are an essential tool that we will use to determine the flow of how our program executes our code

We've already used some conditionals! Think back to our code from last week––we used mouseIsPressed and keyPressed to program specific interactions with our drawing sketch

Booleans

A boolean is type of variable that only has two values : true or false, and will be very important when we start writing conditional logic.

  • 15 is greater than 20 -> false

  • 5 equals 5 -> true

  • 10 is less than or equal than 11 -> true

Combining conditionals

Through combining multiple conditional expressions, we can create complex interactions.

  • && - both conditions have to be true

  • || - only one of the conditions has to be true

Relational Operators

  • < - Less than

  • <= - Less than or equal to

  • > - Greater than

  • >= - Greater than or equal to

  • != - Not equal to

  • == - Equal to

*a note about == vs = , == means checking if something is equal to ... = means saying something is equal to
