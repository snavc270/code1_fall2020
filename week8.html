Week 8: Classes & Objects

What is Procedural Programming?

A coding paradigm in which the problem is broken down into well-defined procedures and steps; how we have been programming.

What is Object Oriented Programming?

A coding paradigm in which data plays the main role. Problems are broken down into objects, the object data, and the functions the objects perform. Think data first!
