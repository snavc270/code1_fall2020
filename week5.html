While Loop

A while loop will run code WHILE a certain condition is true. Once the condition is no longer true, it will exit the loop. A while loop always needs an exit condition! Without an exit, your computer or sketch may crash as the loop runs infinitely.

For Loops

A for loop is another control flow tool that will iterate over a chunk of code a specified number of times.
